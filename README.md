# 🐱 Doraemon's Gadget Hub

A fun and interactive Django web application inspired by Doraemon and his iconic futuristic gadgets.

This project was built while learning Django fundamentals and exploring backend development with Python.

---

## 🚀 Features

- 🏠 Home page with Bootstrap carousel
- 🧰 Gadget showcase with responsive card layout
- 📄 About page
- 📬 Contact page
- 🎨 Clean UI built using Bootstrap 5
- 🔁 Template inheritance using `base.html`
- 📁 Static files integration (images & styling)

---

## 🛠 Tech Stack

- Python 3
- Django
- HTML5
- Bootstrap 5
- Git & GitHub
